# cl_python_2020

## Python scripts for network automation lab

* Clone the git repo
  * git clone https://github.com/jagadnag/cl_python_2020.git
* Navigate to the directory
  * cd cl_python_2020
* Install the python package
  * pip install netmiko
  * pip install napalm
* Verify the python pip packages
  * pip list
* Run the scripts
  * python <script_name>.py
